Paragrafos e quebras de linhas

Você pode escrever um parágrafo de qualquer jeito.
Basta você colocar tudo dentro das tags <p> </p>

Se precisar quebra o texto em algum lugar em especifico você pode usar a tag <br>

Alguns símbolos especias tambem podem ser utilizados por exmplo ao lado
→ para usar essa seta é preciso de &+rarr+;
← para usar a seta a esqueda
↑ para usar seta para cima
↓ para usar a seta para baixo
Você encontra uma tabela com todos os simbolos no pdf da aula 5 :)

Acesse https://www.w3schools.com/charsets/ref_utf_symbols.asp para ver todos os simbolos disponiveis para usar em seus projetos
